# Musée de Monde

Walk into famous paintings. A 3D museum where you physically approach masterpieces on the walls and step inside them as immersive, explorable worlds.

## How It Works

You spawn inside a 3D museum generated by [World Labs Marble](https://worldlabs.ai). Paintings hang on the curved gallery walls. Walk up to one and you seamlessly enter its world — a full 3D Gaussian splat environment generated from the original artwork. Press **X** to return to the museum.

## Tech Stack

- **World Labs Marble** — Generates 500k-point Gaussian splat worlds from 2D paintings
- **SparkJS** (`@sparkjsdev/spark`) — Renders Gaussian splats in the browser
- **Three.js** — 3D rendering pipeline and camera management
- **Next.js** — React framework
- **Pointer Lock API** — First-person mouse look controls

## Architecture

Single-renderer, dual-scene design. The museum and painting worlds are separate `THREE.Scene` objects sharing one `WebGLRenderer` and camera. When you approach a painting (5 units), its splat begins preloading. At 0.7 units, the renderer instantly swaps scenes — no loading screen, no fade.

```
Museum.tsx (orchestrator)
├── Museum scene (museum.spz via SplatMesh)
├── World scene (painting .spz, preloaded on proximity)
├── SparkControls (WASD movement)
└── Pointer Lock (mouse look)
```

## Running Locally

```bash
cd museum-portal
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Click to lock mouse, WASD to walk, mouse to look around.

## Controls

| Key | Action |
|-----|--------|
| **Click** | Lock mouse for free look |
| **WASD** | Walk around |
| **Mouse** | Look around |
| **X** | Return to museum |
| **ESC** | Release mouse lock |
